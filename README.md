## About the Project

> This project is built with React.js, it is a page of fetching API for rockets and missions. On the rockets page you can reserve a rocket, also for the mission you are able to click to join the button in order to join the mission. On the last page My Profile, you can see what all is reserverd.

## Built With

- React.js
- Gitflow
- Redux
- API

### Key Features

- Reserving rockets
- Joining mission
- Seeing your profile
- Reducers
- Redux
- API
- UI design

## Getting Started

Clone this repository to your desired folder:

```sh
  cd your-prefer-folder
  git clone https://github.com/enis-memic/space-travelers
```

### Run this command to run server

> npm start

### Run this command build server

> npm run build

### Run this command to run tests

> npm run test
